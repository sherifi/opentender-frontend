How Opentender Works

Data

Behind Opentender there is robust data collection and processing software, that gathers procurement data from about 20 various sources. Typically, there is no open data, we extract the information from thousands of web pages to get a structured database. After that we group together all publications describing each tender and eventually we create a final most accurate image of particular tender. Source systems and data largely differ from country to country, so tailored piece of software is created for each source separately.

Read more about Challenges, Data processing, Raw data.

Indicators

Opentender uses public procurement performance indicators based on a broad set of scientific analyses, synthesized and taken further in our own research: for transparency indicators see Methods Paper on Fiscal Transparency, for integrity risk indicators see Methods Paper on Corruption Risk Indicators, for administrative capacity indicators see Methods Paper on Administrative Quality Indicators. All of these indicators are based on objective data, are directly actionable by public and private actors, and they are transparently and unambiguously calculated.
